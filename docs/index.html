Затем мы перебираем все ключи в объекте scores с помощью цикла for..in, добавляя значение баллов каждого студента к переменной totalScore. Затем, мы возвращаем итоговое значение totalScore.

getScore1

В данном случае мы объявляем функцию getScore1, которая принимает аргумент scores - объект, содержащий информацию об успеваемости студентов. Мы используем метод Object.values для получения всех значений объекта scores, затем вызываем метод reduce, передавая ему функцию обратного вызова, которая складывает все баллы. В качестве начального значения для reduce мы используем 0, чтобы сумма начиналась с нуля.
